# 🛕 Temples in India

Discover the cultural, historical, and spiritual heritage of India with **Temples in India** — a visually rich web project that showcases ancient and revered temples from across the country.

## 🌟 Live Demo
[(https://rohan-patil11.github.io/Temples_in_India/)](https://rohan-patil11.github.io/Temples_in_India/)

## 📌 Features

- 🌍 Explore famous temples across different Indian states  
- 🔍 Highlighted ancient temples with beautiful images and locations  
- 📖 Temple services: Puja Booking, Tours, Accommodations, and more  
- 🧭 Smooth navigation and modern UI with Swiper sliders  
- 📱 Responsive design for mobile and desktop  
- 🔐 Login & Create account (UI only)  

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling and layout  
- **JavaScript** – Interactivity and animations  
- **Swiper.js** – For featured temple sliders  
- **Font Awesome** – Icons  
- **GitHub Pages** – Hosting  

## 📂 Project Structure

```
Temples_in_India/
├── images/                  # Temple images
├── index.html               # Main HTML file
├── Temples_in_India.css     # Custom styling
├── Temples_in_India.js      # JavaScript interactions
```

## 🚀 How to Run Locally

1. Clone this repo:
   ```bash
   git clone https://github.com/yourusername/Temples_in_India.git
   ```
2. Open `Temples_in_India.html` in your browser.
